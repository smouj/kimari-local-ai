# README Screenshots

This directory contains screenshots used in the main README.

## Naming Convention

Screenshots should follow this naming convention:

- `gateway-overview.png` — Gateway dashboard overview page
- `gateway-chat.png` — Gateway chat playground
- `kimari-console.png` — Kimari console guided setup
- `doctor-deep.png` — Output of `kimari doctor --deep`
- `local-api-health.png` — Health check of the local API
- `integrations.png` — Integration configuration output
- `hf-space.png` — Hugging Face Space screenshot

## Capture Guidelines

See [SAFE_SCREENSHOT_CAPTURE.md](../../SAFE_SCREENSHOT_CAPTURE.md) for the safe screenshot capture guide.

## Rules

- No tokens or API keys visible
- No private paths or email addresses
- No raw training outputs
- No private weights or adapter references
- Gate BLOCKED must be visible if gate state is shown
