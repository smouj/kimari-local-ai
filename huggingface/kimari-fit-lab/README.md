# Kimari Fit Lab

Check if your GPU is compatible with Kimari Local AI.

This Space does **not** run any model. It is a static lookup tool that recommends profiles based on your GPU VRAM.

**Kimari-4B is not released yet.** No weights, adapters, or GGUF files are available.
